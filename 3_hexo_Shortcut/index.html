    Below is the list of shortcuts by for readers from the 3-hexo theme.

    Details

    Global Shortcut

    Key Description
    s show / hide sidebar
    w show / hide post’s table of content
    i focus on search boxes
    j scroll down
    k scroll up
    g page up
    shift+g page down

    Search Box

    Key Description
    esc 1. If there is input in the search box, it will be cleared
    2. if not, remove its focus
    down scroll downward
    up scroll upward
    enter redirect to selected post (default: first search result)
